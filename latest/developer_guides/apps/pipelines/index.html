Creating a Pipeline¶

Figure 29 Creating a pipeline¶

The scheduling component (sched_comp) is configured by the driver inside the IPC request. It is usually set to dai component id for pipelines that are attached to a dai instance and are driven by that dai IO events.

Executing an Operation¶

Most of the pipeline functions sets the operation id and executes a common routine, either component_op_downstream() in case of playback path or component_op_upstream() otherwise.

Scheduling¶

A pipeline’s task (see _Processing_) may be scheduled at certain point in time using pipeline_schedule_copy(start). In order to schedule next stream copy operation in idle (see pre-loader), pipeline_schedule_copy_idle() should be used.
