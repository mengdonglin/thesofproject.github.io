Platform API¶

Location: include/sof/platform.h

group platform_api

Platform API specification.

Defines

PLATFORM_DCACHE_ALIGN¶

Functions

int platform_boot_complete(uint32_t boot_message)¶

Platform specific implementation of the On Boot Complete handler.

Return
0 if successful, error code otherwise.
Parameters
  • boot_message: Boot status code.

int platform_init(struct sof * sof)¶

Platform initialization entry, called during FW initialization.

Return
0 if successful, error code otherwise.
Parameters
  • sof: Context.

static void platform_panic(uint32_t p)¶

Called by the panic handler.

Parameters
  • p: Panic cause, one of SOF_IPC_PANIC_… codes (see ipc.h).
